# TCP/IP

MAC地址是真正负责最终通信的地址，但是由于它不具有层次性，所以IP地址是必不可少的。

IP地址由网络号和主机号两部分组成。

两个设备之间数据流动的物理速率称为**传输速率**，单位bps，又称为带宽。主机之间实际的传输速率被称作**吞吐量**，其单位与带宽相同。吞吐量不仅衡量带宽，同时也衡量主机的CPU处理能力、网络的拥堵程度、报文中数据字段的占有份额（不包含报文首部，只计算数据字段本身）。



**传输速度与计算机内部的表现值**

计算机采用二进制，因此以2<sup>10</sup>表示最接近1000的值。于是有如下等式：

- 1K = 1024
- 1M = 1024K
- 1G = 1024M

而以太网中以*时钟频率*决定传输速度。一下等式请不要与上面混淆

- 1K = 1000
- 1M = 1000K
- 1G = 1000M



**IP地址的标识**

192.168.144.10/24

| 192.168.144 | .10      | /24                            |
| ----------- | -------- | ------------------------------ |
| 网络标识    | 主机标识 | 从头数到第几位为止属于网络标识 |

> 分配IP地址时关于主机标识要注意，用比特位表示主机地址时，不可以全部为0或全部为1。在分配
>
> 过程中应该去掉这两种情况。

**私有网络的IP地址** 

A类： 10.0.0.0 ~ 10.255.255.255 (10/8)

B类： 172.16.0.0 ~ 172.31.255.255 (172.16/12)

C类： 192.168.0.0 ~ 192.168.255.255 (192.168/16)

在此范围之外的IP地址称为全局IP，也叫**公网IP**。

公网IP地址要在整个互联网范围内保持唯一，但私有IP地址只需要在同一个域里保证唯一即可。
